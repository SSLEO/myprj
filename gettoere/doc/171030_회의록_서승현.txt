제목 : 게토리

; 등록한 회원은 자기가 원하는 제목으로 게시판을 개설 및 관리할 수 있는 사이트
; 특정기간(일주일)동안 사용이 적은 게시판(사용 기준은 아직 미정)은 자동적으로 삭제상태가 됨


1. 메인페이지
 - 로고 : ( 이미지 미정 )
 - 로그인, 로그아웃
 - 게시판 검색 : 상단 가운데 배치
 - 신규게시판 : 게시판 검색 하단 우측에 한줄씩 출력
 - 인기글 순위 : (DB조회 방식 미정)
 - 인기게시판 : 하루(또는 현재시간 기준 24시간전)동안 조회 수(또는 글등록 수) 많은 게시판 순위별로 출력
 - 죽은 게시판 : 죽은 게시판 최신순으로 출력


2. 게시판 ( 글 목록, 글 조회, 글 등록, 글 수정 화면 )
 - 게시판 로고이미지
 - 게시판 경고 알림 메시지 : 로고 이미지 또는 게시판 제목 하단에 빨간 글씨로 게시판 사라질수 있는 경고 문구 추가
 - 스마트 에디터 : 글 등록 페이지 에서 스마트 에디터 라이브러리를 받아서 사용
 - 대댓글 : 댓글에 대댓글을 남기는 기능 추가
 - 베댓 : 댓글 중에 추천을 많이 받은 댓글을 댓글목록 상단에 추가 배치( 아직 회의가 안된 기능 )


				------  회의 진행 중 --------

3. 최고 관리자
 - 게시판 검색 : 검색한 게시판을 목록에 출력( 삭제하기 좋게 선택할수 있어야 된다 )
 - 목록 : 검색한 게시판들 목록으로 출력
 - 게시판 삭제 : 목록 내용중 선택하여 삭제를 누르면 게시판이 DB 에서 삭제되는것이 아니라 상태만 죽은 게시판으로 만듦


4. 게시판 관리자 : 계정별로 게시판 관리자 페이지 접속이 가능
 - 게시판 개설 : 게시판 제목 입력 및 로고 이미지 등록가능
 - 부매니저 관리 : 게시판 관리자 페이지에서 부매니저 등록 가능, 등록된 부매니저는 해당 게시판의 글을 삭제할수 있다.
 - 밴 관리 : 게시판 관리자가 특정 닉네임을 자기 게시판에 글을 쓸수 없도록 막을 수 있다.
